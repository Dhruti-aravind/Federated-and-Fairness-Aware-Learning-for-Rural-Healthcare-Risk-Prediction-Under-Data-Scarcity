District-level dataset derived from NFHS-5 (2019–21) covering healthcare, demographic, and socioeconomic indicators across India.
Includes features such as infrastructure access, maternal care, education, and population characteristics.
Designed for machine learning tasks like risk prediction, clustering, and fairness analysis.
Useful for public health research, policy planning, and explainable AI applications.

Link: [https://www.kaggle.com/datasets/dhrutiaravind/nfhs-district-data](https://www.kaggle.com/datasets/dhrutiaravind/nfhs-district-data)
